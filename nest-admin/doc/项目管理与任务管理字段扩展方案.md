# 项目管理与任务管理字段扩展方案

> 起草时间：2026-04-09
> 太子·整理

---

## 一、现状分析

### 1.1 现有字段一览

**Project（项目）— 仅 10 个字段**

| 字段 | 类型 | 说明 |
|------|------|------|
| name | string | 项目名称 |
| code | string | 项目编号 |
| leaderId | string | 项目负责人 |
| startDate | date | 开始时间 |
| endDate | date | 结束时间 |
| status | enum | 项目状态 |
| priority | enum | 优先级 |
| description | text | 描述 |
| attachments | json | 附件 |
| isArchived | char | 是否归档 |

**Task（任务）— 仅 13 个字段**

| 字段 | 类型 | 说明 |
|------|------|------|
| projectId | string | 所属项目 |
| name | string | 任务名称 |
| code | string | 任务编号 |
| leaderId | string | 负责人 |
| executorIds | json | 经办人列表 |
| parentId | string | 父任务 |
| startDate | date | 开始时间 |
| endDate | date | 截止时间 |
| status | enum | 任务状态 |
| priority | enum | 优先级 |
| progress | int | 进度 0-100 |
| description | text | 描述 |
| attachments | json | 附件 |

**ProjectMember — 仅 4 个字段**
projectId, userId, role, isActive

### 1.2 核心问题

1. **项目层**：缺少分类、预算、标签、发起人、所属部门、里程碑、风险等级、质量指标等
2. **任务层**：缺少版本/迭代、预估工时、实际工时、工时记录、任务依赖、优先级细分、审核环节、阻塞标记、工时明细
3. **关联层**：项目-部门关联、角色权限粒度、成员加入时间缺失
4. **流程层**：任务与工作流引擎未打通，无法支撑审批式任务

## 二、方案设计

### 2.1 Project 扩展字段

新增 14 个字段：

```typescript
// ----------- 基础信息扩展 -----------

/** 项目类型：internal=内部项目 external=外部项目 */
@BaseColumn({ type: 'enum', enum: ['internal', 'external'], default: 'internal', name: 'type', comment: '项目类型' })
type: string

/** 所属部门ID */
@BaseColumn({ nullable: true, name: 'department_id', comment: '所属部门ID' })
departmentId: string

/** 项目分类：如"研发"、"市场"、"运营" */
@BaseColumn({ length: 50, nullable: true, name: 'category', comment: '项目分类' })
category: string

/** 项目标签（JSON数组） */
@BaseColumn({ type: 'json', nullable: true, name: 'tags', comment: '项目标签' })
tags: string[]

/** 项目发起人/创建者ID */
@BaseColumn({ nullable: true, name: 'creator_id', comment: '项目发起人ID' })
creatorId: string

// ----------- 时间与周期 -----------

/** 计划开始时间（与实际开始时间区分） */
@BaseColumn({ type: 'datetime', nullable: true, name: 'plan_start_date', comment: '计划开始时间' })
planStartDate: string

/** 计划结束时间 */
@BaseColumn({ type: 'datetime', nullable: true, name: 'plan_end_date', comment: '计划结束时间' })
planEndDate: string

// ----------- 资源与预算 -----------

/** 项目预算（元） */
@BaseColumn({ type: 'decimal', precision: 12, scale: 2, nullable: true, name: 'budget', comment: '项目预算' })
budget: number

/** 币种 */
@BaseColumn({ length: 10, default: 'CNY', name: 'currency', comment: '币种' })
currency: string

// ----------- 里程碑 -----------

/** 里程碑列表（JSON） */
@BaseColumn({ type: 'json', nullable: true, name: 'milestones', comment: '里程碑列表' })
/*
  格式：[{
    id: string,
    name: string,
    planDate: string,
    actualDate: string,
    status: 'pending' | 'completed',
    remark: string
  }]
*/
milestones: MilestoneItem[]

// ----------- 风险与质量 -----------

/** 风险等级：1=低 2=中 3=高 */
@BaseColumn({ type: 'char', length: 1, default: '1', name: 'risk_level', comment: '风险等级' })
riskLevel: string

/** 质量等级：A/B/C/D */
@BaseColumn({ length: 1, nullable: true, name: 'quality_level', comment: '质量等级' })
qualityLevel: string

// ----------- 统计字段 -----------

/** 已消耗工时（小时） */
@BaseColumn({ type: 'decimal', precision: 10, scale: 2, default: 0, name: 'spent_hours', comment: '已消耗工时' })
spentHours: number

/** 客户名称（外部项目用） */
@BaseColumn({ length: 100, nullable: true, name: 'client_name', comment: '客户名称' })
clientName: string
```

### 2.2 Task 扩展字段

新增 15 个字段：

```typescript
// ----------- 版本与迭代 -----------

/** 所属版本/迭代名称 */
@BaseColumn({ length: 50, nullable: true, name: 'version', comment: '版本/迭代' })
version: string

/** 任务所属模块 */
@BaseColumn({ length: 100, nullable: true, name: 'module', comment: '所属模块' })
module: string

/** 任务类型：feature=功能 bug=缺陷 improvement=优化 other=其他 */
@BaseColumn({ type: 'enum', enum: ['feature', 'bug', 'improvement', 'other'], default: 'feature', name: 'task_type', comment: '任务类型' })
taskType: string

// ----------- 工时 -----------

/** 预估工时（小时） */
@BaseColumn({ type: 'decimal', precision: 6, scale: 2, nullable: true, name: 'estimated_hours', comment: '预估工时' })
estimatedHours: number

/** 实际工时（小时） */
@BaseColumn({ type: 'decimal', precision: 6, scale: 2, nullable: true, name: 'actual_hours', comment: '实际工时' })
actualHours: number

// ----------- 依赖 -----------

/** 阻塞任务ID列表（JSON）—— 本任务等待这些任务完成后才能开始 */
@BaseColumn({ type: 'json', nullable: true, name: 'blocked_by', comment: '阻塞本任务的任务' })
blockedBy: string[]

/** 被本任务阻塞的任务ID列表（JSON） */
@BaseColumn({ type: 'json', nullable: true, name: 'blocks', comment: '被本任务阻塞的任务' })
blocks: string[]

// ----------- 审核与评审 -----------

/** 是否需要评审 */
@BaseColumn({ type: 'char', length: 1, default: '0', name: 'need_review', comment: '是否需要评审' })
needReview: string

/** 评审人ID */
@BaseColumn({ nullable: true, name: 'reviewer_id', comment: '评审人ID' })
reviewerId: string

/** 评审状态：pending=待评审 approved=通过 rejected=驳回 */
@BaseColumn({ type: 'char', length: 1, nullable: true, name: 'review_status', comment: '评审状态' })
reviewStatus: string

// ----------- 优先级细分 -----------

/** 紧急程度：1=一般 2=紧急 3=非常紧急 */
@BaseColumn({ type: 'char', length: 1, default: '1', name: 'urgency', comment: '紧急程度' })
urgency: string

// ----------- 标签 -----------

/** 任务标签（JSON） */
@BaseColumn({ type: 'json', nullable: true, name: 'tags', comment: '任务标签' })
tags: string[]

// ----------- 完成标准 -----------

/** 验收标准 */
@BaseColumn({ type: 'text', nullable: true, name: 'acceptance_criteria', comment: '验收标准' })
acceptanceCriteria: string

// ----------- 完成信息 -----------

/** 完成时间 */
@BaseColumn({ type: 'datetime', nullable: true, name: 'completed_time', comment: '完成时间' })
completedTime: string

/** 实际结束时间 */
@BaseColumn({ type: 'datetime', nullable: true, name: 'actual_end_date', comment: '实际结束时间' })
actualEndDate: string
```

## 三、数据库迁移 SQL

```sql
-- ===================== 项目表扩展 =====================
ALTER TABLE `project`
  ADD COLUMN `type` enum('internal','external') NOT NULL DEFAULT 'internal' COMMENT '项目类型' AFTER `description`,
  ADD COLUMN `department_id` bigint DEFAULT NULL COMMENT '所属部门ID' AFTER `type`,
  ADD COLUMN `category` varchar(50) DEFAULT NULL COMMENT '项目分类' AFTER `department_id`,
  ADD COLUMN `tags` json DEFAULT NULL COMMENT '项目标签' AFTER `category`,
  ADD COLUMN `creator_id` bigint DEFAULT NULL COMMENT '项目发起人ID' AFTER `tags`,
  ADD COLUMN `plan_start_date` datetime DEFAULT NULL COMMENT '计划开始时间' AFTER `creator_id`,
  ADD COLUMN `plan_end_date` datetime DEFAULT NULL COMMENT '计划结束时间' AFTER `plan_start_date`,
  ADD COLUMN `budget` decimal(12,2) DEFAULT NULL COMMENT '项目预算' AFTER `plan_end_date`,
  ADD COLUMN `currency` varchar(10) NOT NULL DEFAULT 'CNY' COMMENT '币种' AFTER `budget`,
  ADD COLUMN `milestones` json DEFAULT NULL COMMENT '里程碑列表' AFTER `currency`,
  ADD COLUMN `risk_level` char(1) NOT NULL DEFAULT '1' COMMENT '风险等级' AFTER `milestones`,
  ADD COLUMN `quality_level` char(1) DEFAULT NULL COMMENT '质量等级' AFTER `risk_level`,
  ADD COLUMN `spent_hours` decimal(10,2) NOT NULL DEFAULT 0.00 COMMENT '已消耗工时' AFTER `quality_level`,
  ADD COLUMN `client_name` varchar(100) DEFAULT NULL COMMENT '客户名称' AFTER `spent_hours`;

-- ===================== 任务表扩展 =====================
ALTER TABLE `task`
  ADD COLUMN `version` varchar(50) DEFAULT NULL COMMENT '版本/迭代' AFTER `status`,
  ADD COLUMN `module` varchar(100) DEFAULT NULL COMMENT '所属模块' AFTER `version`,
  ADD COLUMN `task_type` enum('feature','bug','improvement','other') NOT NULL DEFAULT 'feature' COMMENT '任务类型' AFTER `module`,
  ADD COLUMN `estimated_hours` decimal(6,2) DEFAULT NULL COMMENT '预估工时' AFTER `task_type`,
  ADD COLUMN `actual_hours` decimal(6,2) DEFAULT NULL COMMENT '实际工时' AFTER `estimated_hours`,
  ADD COLUMN `blocked_by` json DEFAULT NULL COMMENT '阻塞本任务的任务ID列表' AFTER `actual_hours`,
  ADD COLUMN `blocks` json DEFAULT NULL COMMENT '被本任务阻塞的任务ID列表' AFTER `blocked_by`,
  ADD COLUMN `need_review` char(1) NOT NULL DEFAULT '0' COMMENT '是否需要评审' AFTER `blocks`,
  ADD COLUMN `reviewer_id` bigint DEFAULT NULL COMMENT '评审人ID' AFTER `need_review`,
  ADD COLUMN `review_status` char(1) DEFAULT NULL COMMENT '评审状态' AFTER `reviewer_id`,
  ADD COLUMN `urgency` char(1) NOT NULL DEFAULT '1' COMMENT '紧急程度' AFTER `review_status`,
  ADD COLUMN `tags` json DEFAULT NULL COMMENT '任务标签' AFTER `urgency`,
  ADD COLUMN `acceptance_criteria` text DEFAULT NULL COMMENT '验收标准' AFTER `tags`,
  ADD COLUMN `completed_time` datetime DEFAULT NULL COMMENT '完成时间' AFTER `acceptance_criteria`,
  ADD COLUMN `actual_end_date` datetime DEFAULT NULL COMMENT '实际结束时间' AFTER `completed_time`;

-- ===================== 项目成员表扩展 =====================
ALTER TABLE `project_member`
  ADD COLUMN `joined_time` datetime DEFAULT NULL COMMENT '加入时间' AFTER `is_active`,
  ADD COLUMN `department_id` bigint DEFAULT NULL COMMENT '所属部门ID' AFTER `joined_time`,
  ADD COLUMN `job_title` varchar(50) DEFAULT NULL COMMENT '工作角色' AFTER `department_id`,
  ADD COLUMN `notify_enabled` char(1) NOT NULL DEFAULT '1' COMMENT '是否接收通知' AFTER `job_title`;
```
